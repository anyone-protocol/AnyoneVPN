# Anyone VPN

### Android Anon VPN

Anyone VPN is a freely licensed open-source application developed for the
Android platform. It acts as a front-end for the anyone-protocol binary application,
while also providing a secure HTTP Proxy for connecting web browsers and other
HTTP client applications into the Tor SOCKS interface.

### Build Instructions

```sh
git clone --recursive https://github.com/anyone-protocol/AnyoneVpn.git
```

Or, if you already cloned the repo:

```sh
cd AnyoneVpn
git pull
git submodule update --init --recursive
```
